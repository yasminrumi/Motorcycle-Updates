# 🏍️ Motorcycle Sorting Visualizer

A web-based tool to visualize sorting algorithms with a fun twist—a motorcycle moving across the bars! 

## 🛠️ Built With:
* **HTML5**: For the structure of the visualization area.
* **CSS3**: For styling the bars and the animated motorcycle.
* **JavaScript**: For the sorting logic and movement animations.

## 🚀 Features:
- Visual representation of data sorting.
- Custom motorcycle icon as a visual pointer.
- Smooth CSS transitions for realistic movement.

## 📂 Project Structure:
- `index.html`: The main entry point.
- `style.css`: Contains the layout and motorcycle animations.
- `script.js`: The core logic for sorting and interactivity.
